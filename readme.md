# 基于springboot的新闻管理系统5

## 1、项目介绍

基于springboot的新闻管理系统5拥有两种角色：管理员和用户

管理员：新闻管理、类型管理、留言管理、评论管理、用户管理等

用户：查看新闻、留言、收藏、登录注册


## 2、项目技术

后端框架：springboot、Mybatis

前端技术：Bootstrap、jsp、css、JavaScript、JQuery、VUE

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：不需要
- 数据库类型：MySql（5.5-5.7、8.x版本都可） 
- maven版本：无限制
- 硬件环境：Windows
